jquery.a11yTree

A very lightweight, accessible, and uninvasive jQuery plugin that transforms nested lists into an expanding and collapsing tree.

The Idea

Adding functionality to expand and collapse tree nodes and their nested children is fairly trivial. Unfortunately, many implementations don't include the functionality people using assistive technologies depend on, or are just more complex than what's needed.

Inspirations

DynaTree and Fancytree jQuery plugins
They're great—I just wanted something simpler, and less invasive to the the original DOM.
(Not so) Simple ARIA Tree Views and Screen Readers
This is a great article that breaks down all of the ARIA roles and the additional semantics ARIA provides to a document.

Navigation

a11yTree provides mouse navigation via the expand and collapse toggles, as well as the following tree keyboard navigation:
down
navigates down to the next expanded tree node
up
navigates up to the previous expanded tree node
right
expands the tree node in focus if it's collapsed and has children, or navigates to the first child node of an expanded parent
left
collapses the tree node in focus if it's expanded, or navigates to the parent node

How it Works

Include a list of nested lists in your DOM that you wish to turn into a tree or an existing tree that you'd like to add additional ARIA semantics to and keyboard navigation.
  • parent 1
    • child 1
      • grandchild 1
      • grandchild 2
    • child 2
  • parent 2
    • child 1
    • child 2
      • grandchild 1
      • grandchild 2
Now apply a11yTree with minimal options to the parent list element.
$('ul.code-example').a11yTree();
And you get this:
  • parent 1
    • child 1
      • grandchild 1
      • grandchild 2
    • child 2
  • parent 2
    • child 1
    • child 2
      • grandchild 1
      • grandchild 2
